Getting an ASN and a Prefix

To run your own ISP, you will need a few things, mainly internet resources from a LIR and internet transit from a provider.

• ASN
You can read more about what an ASN is in our previous article here. In short, it's an identifying number for your network.
You can start the process of getting an ASN here, and it will usually take 2-3 business days for yours to be issued.

• IP Prefix
IP Prefix is a group of IP addresses that you can own or rent. You can start the process here, it is very quick and easy, and you should usually get it within one business day.

• Server Provider (Optional if you're in the RIPE Service Region
If you're in the RIPE service region, you can ignore this step if you want to set up your network from your home. But you can also set it up from a server somewhere else like a cloud provider.
If you're not in the RIPE service region, you will need to have a server in the RIPE region. Below is a list of server providers in the RIPE region that also provides transit.
• Vultr
• Hubflake
• VMHaus

• Transit Provider
You will need two networks to upstream your network to access the rest of the internet and have justification for an ASN. Below are a few free transit providers that I recommend. If you live in the RIPE service region, you can run your whole network directly from your home and connect to two of the transit providers below.
• EricNet
• Freetransit
• Inferno
• tunnelbroker.li
• Route48
